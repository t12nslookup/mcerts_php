How to add an analyte to a reference material

1. Start from the MCERTS main menu:

2. In the 'Reference Materials' column click [view] next to the validation you wish to add a reference material to.

3. Click [view] in the Analyte column for the reference material you wish to add an analyte to.

4. Click [add analytes to reference material]

5. You are now presented with a list of analytes. Tick the boxes next to the analytes you wish to add and enter the target concentration for each analyte.

6. Click the "Add Checked Compounds" button.
