How to add a reference material to a validation report

1. Start from the MCERTS main menu:

2. In the 'Reference Materials' column click [view] next to the validation you wish to add a reference material to.

3. Click [add new reference material]

4. Choose the reference material from the drop down menu.

Enter a trivial name for the material (if there is one)

Enter its default standard deviation and bias

5. Click the "Add Reference Material" button
