Add a validation report

1. From the main menu (as shown above) click [add validation report], this will bring you to this screen :

2. Choose the equipment id from the drop down list.

3. Enter a suitable validation name. Previous names include : '625 SVOC Data', 'Manchester Metals' and '645 VOC Data'

4. Click the submit button to add the validation. The validation will now be visible from the mcerts main menu (top picture)
