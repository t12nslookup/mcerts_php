ICP Expert Software (Vista machines)

Setting up the Machine

The *.sws files that the machine creates must be saved into a folder with the name EIDXXX, where XXX is the equipment id of the machine.

(The EIDXXX folder does not have to be the immediate parent folder of the save location, i.e C:\EID345\DATA\1 is acceptable)

To do this a new folder on the computer must be created. It can be created anywhere on the computer.

E.g. For each run follow these steps:

1. Click File

2. Click Save As

3. Choose a folder C:\DATA\EIDXXX\ (replace XXX with equipment id)

4. Click Save.

 

To Export the data required by MCERTS

The ICP machine exports data as a CSV file. Every time you export data from the machine you must follow these instructions :

1.Click on the column heading labelled “Sample Labels” (as shown above). This should select all the data for the current run.

2.Click File on the top menu

3.Click Export Settings

4.Tick the boxes next to “Worksheet name” and “Worksheet full path”

5. Enter a filename in the destination filename box. This is the file that you will upload into the MCERTS system

6. Click the Browse button and choose a folder to save the file into.

7.Click the “Export Now” button, then click the “OK” button.

The file in the Destination Filename text box is the file that you will upload to the MCERTS system.
