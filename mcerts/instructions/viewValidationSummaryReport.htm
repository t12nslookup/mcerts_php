How to view a validation summary report

There are 2 types of summary:

1. Pass/fail summary : this summary shows passes and fails for each analyte in each reference material.

2. MCERTS Summary : this summary is a more detailed version of the pass/fail summary report. It shows the actual bias and precision values for each analyte. This is the summary that MCERTS require from us.

A summary report must be created before it can be viewed. Also whenever the data in a validation is changed (ie. new batches added, batch selection alters or new data added) then a summary must be recreated.

From the main screen :

Create pass/fail summary report

Clicking [create test] under the 'Pass/Fail Summary' column creates a summary that only SAI/SAL people can see.

Clicking [create final] under the 'Pass/Fail Summary' column creates a summary that UCAS/MCERTS will see.

View pass/fail summary report

Click [view test] or [view final] to see the summary report.

Create MCERTS summary report

Clicking [create] under the 'MCERTS Summary' column to create an MCERTS summary

View MCERTS summary report

Click [view] under the 'MCERTS Summary' column to view the summary.
